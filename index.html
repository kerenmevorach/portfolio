  • Googlie

  • Product Design

  • 2023

Googlie is a crafting marketplace app that allows users to borrow and lend crafting supplies. The app aims to inspire crafters and provide them with the opportunity to explore new passions without committing to purchasing new equipment.

This project involved conducting research, scoping, and designing and testing low-fi, mid-fid, and hi-fi prototypes.

Reach out to view the full case study and prototype.

Branding
Googlie is all about exploring, whether it‚Äôs through discovering new hobbies, searching for materials, or finding new passions. This idea of exploration and searching is represented through googlie eyes.

Borrowing Supplies from Crafters
Googlie gives crafters the opportunity to borrow crafting supplies from others. Users can select a craft category, and use filters to narrow their search. The user can then pick which item they are interested in, see product details, and send the lender a message to borrow an item.

Expanded view of crafting categories for users to browse.

Collapsed view of crafting categories.

Once users select a crafting category, they can view products and use filters to further refine their search.

Once the user selects an item, they can view images, product details, and send a message to the lender.

Lending Supplies to Crafters
Users can also list items to lend to borrowers. Lenders will add product details, upload photos, set a pickup address, and set a price for the product.

Users can add a product name, description, select the condition and assign it to 1‚Äì3 categories.

The user will then upload photos of the product.

The user will then set the pickup location for the product.

The user will then set a price for the item. They can set a daily, weekly, and/or a monthly rate.

The journey of a borrower searching for crafting materials, and a lender making a product listing.
